# Kujob

Minimalist all-in-one job queue with support for multiple providers.
Drivers provided : 
- PostgreSQL
- MySQL
- Redis
- InMemory

# Features

- Schedule & Run jobs
- Group jobs into queues
- Prioritize jobs within a queue
- Retry failed jobs a fixed number of times
- Delayed Jobs
- Custom Scheduling
- Configure back-off for failed jobs
